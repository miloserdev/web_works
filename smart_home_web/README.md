A smart home web application for personal use  
Also check experiments/esp32_relay  

## Features
* Devices
  * Specify command per pin of MCU
  * HTTP/HTTPS control
  * Status monitoring HTTP/HTTPS/WebSocket
* Rooms
  * Control devices and buttons per room
  * Set scene per room
* Scenes
  * // I am working about it
* Automations
  * Setup automations for all devices including root broker
  * Exec command (run by time / run after timeout (if power turned on))
  * Exec after packet received // Now in develop
